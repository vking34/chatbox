# CZ Messenger

## Techstack
- Nodejs: Runtime Environment
- Express: Web framework
- Socket.io: Socket library.
- Redis (for scalability): pub/sub messages and socket ids
- MongoDB: Store message

## Namespace: /messenger

## Room:
1. Room type:
     - ```BS```: Buyer.Seller
     - ...

2. Room ID: BuyerID.SellerID

3. Buyer: BuyerID
4. Seller: SellerID
